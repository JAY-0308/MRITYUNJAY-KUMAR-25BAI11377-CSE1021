A Personal Finance Management (PFM) App

Overview of the project :
A utility app to help users to manage their money, a real-world problem for most consumers to track their expenses and transsactions.

Problem: Individuals struggle to track spending, stick to budgets, and identify areas for saving.

Database Design: Structure for users, accounts, transactions, and categories.

Feature: Implement expense categorization (e.g., 'Groceries', 'Utilities') and a budget tracker that alerts the user when they approach their spending limit.

Technology: Python using math module , arrays ,built-in function and loops.
